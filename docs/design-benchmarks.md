# Design Benchmarks

The BINGE visual direction is **European Architectural Systems**: technical credibility, calm editorial imagery, and clear product discovery.

## Reference pattern mix

- Schlüter-Systems: system-led information architecture and technical trust
- Progress Profiles: broad profile taxonomy, dimensions, finishes, and download structure
- Intrim Australia: warm project imagery, style-led discovery, and sample journeys
- Döllken Profiles: specialist skirting tools and installation guidance

## Principles

- Do not copy another brand's visual identity or page composition.
- Use large, quiet architectural imagery with captions outside the image.
- Make product codes, installation types, dimensions, and downloads easy to scan.
- Separate inspiration from technical evidence.
- Keep “Request a Quote”, “Request a Sample”, and “Download Technical Drawing” visible.
- Avoid marketplace badges, public retail pricing, crowded cards, fake certifications, and generic SaaS styling.
