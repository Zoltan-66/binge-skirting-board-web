# Product specification approval workbook

Use this checklist before publishing exact specifications or preparing quotations. Until a row is approved, the website should keep the wording “available on request” or “confirmed per project”.

## Required fields per product family

| Field | What to confirm |
| --- | --- |
| Product code | Final factory code and export-facing code |
| English name | Approved sales name |
| Chinese source name | Factory name used in drawings and production |
| Material | Alloy, steel grade, timber species or WPC composition |
| Profile dimensions | Height × depth × wall thickness |
| Standard length | Stock lengths and custom-length limits |
| Surface finish | Process, colour range, gloss and texture |
| Installation system | Clip, adhesive, screw, recessed base or other method |
| Accessories | Corners, end caps, joints, clips and fasteners |
| Packing | Pieces per carton, carton size and gross/net weight |
| MOQ | Standard finish and custom finish MOQ |
| Lead time | Sample and production lead time |
| Compliance | Available reports, standards and report numbers |
| Drawing | Approved PDF/DWG version and revision date |
| Quote basis | EXW/FOB basis and quotation unit |

## Website publication gate

1. Factory confirms dimensions and material.
2. Sales confirms the English name and target market wording.
3. Engineering approves the drawing revision.
4. Export team verifies compliance wording against actual reports.
5. Product record changes from `Awaiting factory confirmation` to approved.
