# Product Content Model

Each public product series requires:

- Stable series code
- English product name
- URL slug
- Category and material family
- Installation method
- Short description
- Verified feature list
- Known sizes and finishes
- Compatible accessories
- Application sectors
- Product gallery with approved alt text
- Profile drawing and technical downloads
- Claim verification state
- RFQ availability

Verification states:

- `verified`: supported by an approved drawing, report, or factory record
- `provisional`: acceptable for internal review, not public
- `omitted`: intentionally excluded until evidence exists

Original Chinese e-commerce artwork is source material only. Public images must remove marketplace badges, retail pricing, unsupported claims, and Chinese promotional overlays unless a bilingual asset is intentionally approved.
